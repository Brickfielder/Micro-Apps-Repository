Pure GitHub Pages version — pulls public RSS via a CORS-friendly proxy (no API keys).

Choose a topic and click “Load Latest & Build Quiz”.
About this demo
  • No backend. The browser fetches RSS using api.allorigins.win to avoid CORS issues.
  • Client-side cache (localStorage) for 6 hours to reduce load on publishers.
  • Varied question types + 5 “real vs fake” headline items.
Built for educational purposes. Always verify via the linked sources.
